**Frontend:**

  - [GitHub Repository](https://github.com/SYYYanyangyu/Noodles.LearnEnFront/tree/master)

- Frontend UI:
  - React + TypeScript

**Backend:**

- C#/NET8

- RabbitMQ + MediatR
